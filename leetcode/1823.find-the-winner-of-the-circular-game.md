### Simulation
```kotlin
fun findTheWinner(n: Int, k: Int): Int {
    val queue = ArrayDeque<Int>()
    for (i in 1..n) {
        queue.addLast(i)
    }
    var count = 1
    while (queue.size > 1) {
        while (count < k) {
            queue.addLast(queue.removeFirst())
            count++
        }
        // Remove the k-th element
        queue.removeFirst()
        count = 1
    }
    return queue.removeFirst()
}
```

* **Time Complexity**: `O(n * k)`.
* **Space Complexity**: `O(n)`.
